What is ADG?

Platform overview

The Agentic Development Platform is a small, local control plane that fixes two failure modes that dominate AI-assisted development.

Problem 1 — Unbounded, unaudited actions

Hand an AI agent your codebase and it does too much: destructive commands, policy-bypasses, work that cannot be audited or rolled back. ADG makes every tool map to a risk class; by default everything is denied unless explicitly permitted.

Problem 2 — Context window overflow

You paste the whole tracker JSON and burn six figures of tokens before the agent writes a line. ADG's context broker refuses to load the artifacts that blow up a context window and returns a few-KB bounded packet instead.

The platform in one line: task → classify → SQL lookup → capped context packet → anchored files → targeted checks

Five design pillars

Governance model

1. Generated artifacts are hazards

Full SQL dumps, JSON mirrors, and generated HTML go on a forbiddenBulkFiles denylist. SQLite selects the context instead of dumping it.

2. Governance for agents, not auditors

Audit trails, DORA metrics, and decision events with reason/risk/rollback exist so that a non-human collaborator's work is non-repudiable — even solo.

3. Threat-model your own agents

Tools default to deny. Prompt-injection, cross-scope escalation, and excessive-agency are standing assumptions, not reactions. Agents are untrusted insiders.

4. Append-only by construction

The audit log is event-sourced and never rewritten. Current state (feature status, item lifecycle) is always derived from events via SQL views.

5. Deliberate restraint

node + sqlite3 + JSONL. No SaaS, no vector database, no agent framework. Everything is grep-able, diffable, and offline. The full gate runs in seconds.

Quick start

Setup

Requirements: Node ≥ 20 and the sqlite3 CLI on PATH. No npm install needed — there are no runtime dependencies.

npm run setup          # build data/backlog.sqlite from schema + seed + audit log
npm run ci:governance  # full gate: backlog, audit, guardrails, evals, dora, broker
