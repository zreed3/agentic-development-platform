# bord.room Context Primer

Use this before reading broad docs or editing.

## Loop

1. Choose the feature ID from the user request or SQLite.
2. Generate one bounded packet:
   ```sh
   pnpm context:lite -- --feature S30
   ```
3. Read only packet files unless local evidence points elsewhere.
4. Name the slice: feature, route/action, persona, scope, risk class, files, and checks.
5. Stop if the feature ID, authority boundary, or evidence path is unclear.

## Output

Return a compact context brief:

- feature and item IDs;
- exact files in scope;
- permissions, entitlement, tenant scope, and business scope;
- tests/gates to run;
- stop conditions.
