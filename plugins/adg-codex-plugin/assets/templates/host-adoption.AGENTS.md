# ADG Codex Plugin Host Adoption

Add this discipline to a host repo's agent instructions:

- Start material work from a bounded ADG context slice.
- Run bundled plugin commands from the governed host repo as the working directory.
- Keep requirements-to-UX lineage queryable before implementation.
- Treat UI visibility as separate from authorization.
- Record material deliverables with source inputs, graph slice, files touched, tests, decisions, failures, and evidence.
- Use standards-map controls as local evidence anchors, not as copied standard text.
